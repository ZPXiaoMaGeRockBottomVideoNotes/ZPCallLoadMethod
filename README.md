# ZPCallLoadMethod
本Demo主要介绍了类中的load类方法在什么时候加载，还介绍了load类方法的调用原理，还重点介绍了load类方法调用顺序的原则。除此之外，还复习了类对象调用分类中的方法的原理，看起来像是分类里面的方法把原类里面的方法给覆盖了。
